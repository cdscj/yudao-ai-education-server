**技术栈**：Spring Boot 3.x、Spring Cloud Alibaba、Spring Security OAuth2、MyBatis-Plus、MySQL、Redis、Nacos、Knife4j

- **开发规范实践**：遵循 DDD 分层架构（Controller → Service → DAL），使用 MapStruct 进行 VO/DO/DTO 对象转换，统一 `CommonResult` 响应封装与 `ServiceException` 异常体系，熟练企业级后台开发规范
